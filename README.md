# Anomaly Detection
A flask and a machine learning-based application for anomaly detection in
financial transactions

## Project Overview
1. *Flask for a web application and API*
2. *html,css*
3. Exploratory Data Analysis
4. *Machine learning algorithms*:-
   1. *Isolation Forest*
   2. *K-Means*
   3. *DBSCAN* (Here, outliers are anomalies)
